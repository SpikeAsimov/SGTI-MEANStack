Guía de uso

1 - Elegí el tipo de trámite que queres realizar

Podés elegir entre diferentes tipos de trámites

2 - Completa el formulario

Debes rellenar los campos con los datos necesarios para gestionar el trámite elegido.

3 - Confirmá los datos

Revisá los datos ingresados, y el medio de pago.

4 - Elegí el medio de pago

Podes realizar el pago a través de diferentes opciones.

5 - Seguí el estado de tu trámite

Una vez confirmado el proceso, podés realizar el seguimiento del mismo a través de la opción Mis trámites.
